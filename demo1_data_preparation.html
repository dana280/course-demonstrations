Essential steps before statistical analysis

1Checking Data Completeness

What we check:

  • Questions with excessive missing responses: Identify questions where more than 33% of respondents didn't answer
  • Incomplete respondents: Find participants who didn't answer most questions or skipped important demographic items
  • Decision criteria: Consider removing problematic questions or respondents from the dataset
üéØ Goal: Ensure data quality and completeness before proceeding with analysis
Example: If Question 5 has only 10 out of 150 responses (93% missing), it may be too problematic to include in the analysis.
2Handling Missing Data & Response Distribution

Actions performed:

  • Check frequency distribution: Examine how responses are distributed across each question's scale
  • Identify outliers: Find values that don't match the expected response scale
  • Define missing values: Mark "not relevant," "don't know," or "other" responses as missing data
  • Verify data format: Ensure all responses are in the appropriate format for calculations (e.g., numerical for means)
üìä Statistical Note: Missing data should be properly coded to ensure they don't affect mean calculations or statistical tests
Example: In a 1-5 satisfaction scale, code "Not Applicable" as 99 and define it as a missing value, so it won't be included when calculating the average satisfaction score.
3Creating New Variables

Category reduction process:

  • Original scale: 5-point Likert scale (1 = Not at all, 5 = Very much)
  • New simplified scale: 3-point categorization
  • Recoding scheme:
    • Categories 1 + 2 ‚Üí 1 = Low evaluation
    • Category 3 ‚Üí 2 = Medium evaluation
    • Categories 4 + 5 ‚Üí 3 = High evaluation
‚ö†Ô∏è Important: New variables (e.g., Q1_new, Q2_new) are added to the file without overwriting the original 5-point scale variables. Both versions remain available for different types of analyses.
When to use each version:
  • 5-point scale: For calculating means and standard deviations
  • 3-point scale: For frequency distributions and cross-tabulations
